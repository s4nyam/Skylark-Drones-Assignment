Run in command line  : python app.py and you ll get an output file called "Mapi.html"


Also Please install dependencies before running the app:

"sudo pip install pandas"    -           to read CSV file

"sudo pip install folium"      -         to gain leaflet.js power in python
folium builds on the data wrangling strengths of the Python ecosystem and the mapping strengths of the leaflet.js library. Manipulate your data in Python, then visualize it in on a Leaflet map via folium .


also install all the tools required for data science in python :

Numpy + Scipy + Matplotlib + plotly + pandas + jupyter + seaborn
